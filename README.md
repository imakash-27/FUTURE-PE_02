# FUTURE_PE_02 — AI Content Marketing using UGC Ads

**Internship Track:** Prompt Engineering (PE) — Future Interns
**Task:** Generate an AI-powered UGC ad content pack for a real product/business
**Business:** Prime Relocations — premium packers & movers, Nellore, Andhra Pradesh
**Tools Used:** Claude (claude.ai)

## Overview
This repository documents the creation of a UGC-style ad content pack —
scroll-stopping hooks, full ad scripts, and platform-specific captions —
built through structured, constraint-driven prompting to sound like real
customer content rather than brand marketing copy.

## Skills Demonstrated
- AI-driven content marketing
- UGC ad scripting and hook writing
- Ad psychology and conversion-focused messaging
- Platform-specific tone adaptation (Instagram, YouTube Shorts, Facebook)

## What's Inside
- `deliverable.md` — 6 hooks, 3 full ad scripts, captions, and CTAs
- Documented prompt sequence (persona/constraints → hooks → full scripts)
- Notes on why each prompt decision was made

## Process Summary
1. Set UGC persona constraints (first-person, no corporate language)
2. Generated 6 scroll-stopping hooks across different angles
3. Built 3 full platform-specific scripts from the strongest hooks
4. Added captions, hashtags, and CTAs tailored to each platform
